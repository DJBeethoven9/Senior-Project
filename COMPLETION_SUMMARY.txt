 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

STUCK ON "CALIBRATING":
  → Check serial output for errors
  → Verify WiFi connected
  → Restart ESP32
  → See TROUBLESHOOTING.md #1

NEVER DETECTS HUMANS:
  → Decrease MOTION_THRESHOLD
  → Move around more during monitoring
  → Check CSI data flows (dashboard graphs)
  → See TROUBLESHOOTING.md #2

FALSE ALARMS:
  → Increase MOTION_THRESHOLD
  → Turn off fans/AC during calibration
  → Increase MIN_DETECTION_FRAMES
  → See TROUBLESHOOTING.md #3

DASHBOARD NOT LOADING:
  → Check WiFi connection
  → Verify IP: http://192.168.4.1
  → Try hard refresh: Ctrl+Shift+R
  → See TROUBLESHOOTING.md #10

COMPILATION ERROR:
  → Run: platformio clean
  → Rebuild: platformio run -t upload
  → See TROUBLESHOOTING.md #6
